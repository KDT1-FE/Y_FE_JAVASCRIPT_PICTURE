메인 메뉴

야놀자는 누구나 마음 편히 놀 수 있게 여가의 패러다임을 전환합니다.
세상의 모든 여행 Yanolja App만 있으면 누구나 언제 어디서든 여행을 즐길 수 있습니다.

  • 회사정보

    야놀자에 대해서 알아볼 수 있는 페이지를 제공하고 있습니다.
    Provides a page to learn about Yanolja.

  • 직원사진관리

    직원들의 사진을 관리하는 서비스 페이지를 제공하고 있습니다.
    We provide a service page for managing employees' photos.

  • 회사캘린더

    야놀자의 스케줄을 볼 수 있는 페이지를 제공하고 있습니다.
    It provides a page where you can see Yanolja's schedule.

  • 회사위치

    야놀자(회사)의 위치 정보를 볼 수 있는 페이지를 제공하고 있습니다.
    Provides a page for viewing Yanolja location information.
