# ✨ 계곡 포인트 사진 관리 서비스

---

### ✅ 개요

- 계곡에서 놀 수 있는 포인트를 관리하는 서비스를 만든다.
- Firebase Storage를 통해서 cors를 해본다.
- 반응형 웹과 애니메이션을 구현해본다.
