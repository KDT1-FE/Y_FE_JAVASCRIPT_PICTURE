# :camera: 프로젝트 매니지먼트 서비스

## [필수 요구사항]

- “AWS S3 / Firebase 같은 서비스”를 이용하여 사진을 관리할 수 있는 페이지를 구현하세요.
- 프로필 페이지를 개발하세요.
- 스크롤이 가능한 형태의 리스팅 페이지를 개발하세요.
- 전체 페이지 데스크탑-모바일 반응형 페이지를 개발하세요.
- 사진을 등록, 수정, 삭제가 가능해야 합니다.
- 유저 플로우를 제작하여 리드미에 추가하세요.

* CSS
  - 애니메이션 구현
  - 상대수치 사용(rem, em)
* JavaScript
  - DOM event 조작
