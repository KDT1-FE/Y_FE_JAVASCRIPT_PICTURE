## AWS Amplify 
기본적인 CRUD및 웹 호스팅을 Amplify를 통해 간편하게 생성 가능.

## Webpack
HMR을 이용한 개발 속도 증대 및 번들링을 통한 로딩 속도 단축

## SPA
바닐라 JS를 이용해 컴포넌트 및 해쉬 라우터를 구현하여 SPA 구현
